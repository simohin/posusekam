# ADR 001: Базовая инфраструктура (Homelab)

## Статус
Принято и реализовано

## Контекст
Проект «По сусекам» разворачивается на выделенном сервере (Mac mini) под управлением Ubuntu Linux. Необходимо обеспечить изолированную, надежную и масштабируемую инфраструктуру в концепции Enterprise Cloud-Native.

## Решение
- **Оркестратор:** K3s (Lightweight Kubernetes). Развернут с внешним доступом через `kubectl`.
- **База данных:** PostgreSQL 16. База развернута на хост-машине (вне кластера K3s) для производительности хранения. Для приложения создана выделенная БД `posusekam` и отдельный пользователь-владелец с ограниченными правами.
- **Управление секретами:** HashiCorp Vault развернут внутри K3s. Используется движок kv-v2. Аутентификация микросервисов (подов) осуществляется через встроенный метод `kubernetes` на основе Service Accounts. Политики Vault четко ограничивают доступ каждого сервиса только к своим путям (например, `secret/data/posusekam/database`).
